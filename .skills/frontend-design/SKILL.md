# Frontend Design

## Summary

Use this skill when a page needs stronger structure, hierarchy, spacing, and responsive behavior.

## Steps

1. Clarify the primary action and make sure the layout supports that action first.
2. Establish a predictable information hierarchy with strong sectioning, spacing, and contrast.
3. Make mobile behavior explicit instead of assuming desktop layout will collapse well.
4. Keep component structure simple enough that generated code remains editable.
