# Build Map Website

## Summary

Use this skill when the task is a map-first trail or location website that needs SDK isolation and responsive UI.

## Steps

1. Check current map SDK guidance and compare at least one free and one managed option.
2. Keep tokens and SDK setup inside a dedicated config or wrapper file.
3. Design mobile-first layout around the map viewport, filter controls, and result cards.
4. Add a small README section describing local preview and API key setup.
